Functionality Goals:

Display a weekly calendar, where a user would be able to add events to the calendar, remove events, and save their personalized calendar to a server and retrieve it at any time. Ideally, we will be able to implement session management to save each unique schedule to an associated user. Additionally, we would also like to implement roles to specific users: Users and Managers, where the managers would be able to view any employees saved schedule and approve of a user's schedule.
